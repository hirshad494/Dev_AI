# DevAI - Your Intelligent Programming Assistant

**DevAI** is an AI-powered application that brings the power of OpenAI's cutting-edge machine learning model to your fingertips. With an elegant user interface resembling popular chat applications, DevAI offers advanced communication with the GPT-3 model API. Most importantly, it provides the ability to ask the AI for help with a wide range of topics, not limited to programming. Whether you need assistance with translation, generating content, or answering questions, DevAI has got you covered.

**Check out the live application: [DevAI on Vercel](https://devai-coral.vercel.app/)**

## Key Features

- **Cutting-Edge AI**: DevAI leverages OpenAI's state-of-the-art machine learning model to provide intelligent responses and assistance on various topics.

- **User-Friendly Interface**: The application's interface is designed to resemble popular chat applications, ensuring a familiar and intuitive experience.

- **Multi-Purpose Support**: DevAI is not just for programming! It can assist with a wide array of tasks, from language translation to generating content and answering questions.

- **Integration of Modern Technologies**: DevAI is built using the most in-demand technologies today, including HTML, CSS, JavaScript, Vue.js, and Node.js.

## Technologies Used

DevAI is a vanilla JavaScript project powered by Vue.js on the client-side and Node.js on the server-side. Here are some of the technologies and features that make this application stand out:

- **Responsive Layouts**: DevAI uses pure CSS to create responsive layouts, ensuring an optimal user experience on all devices.

- **Event Handling**: JavaScript is employed to handle various user interactions and events within the application.

- **Markdown Rendering**: DevAI can render Markdown content from strings, making it easy to display formatted text and code snippets.

- **Code Highlighting**: The application uses Prisma for code highlighting, enhancing the readability of code snippets.

- **API Requests**: DevAI can make API requests to external services, expanding its capabilities and usefulness.

- **Error Handling**: The codebase includes robust error handling to ensure smooth and reliable performance.

- **Clean Code**: DevAI's codebase adheres to best practices, making it maintainable and extensible.

## Getting Started

To start using DevAI and harness the power of AI-assisted tasks, follow these simple steps:

1. Clone the repository to your local machine.

2. Install the necessary dependencies using npm or yarn.

3. Launch the Node.js server to run the application.

4. Open your web browser and navigate to the provided URL.

## Usage

DevAI can be used for a wide range of tasks, including:

- Seeking assistance and explanations.
- Language translation.

- Answering questions on various topics.
- Exploring AI-driven solutions for diverse challenges.
